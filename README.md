# 🎮 Jolt Time - Історична Тапалка

Повноцінна історична тапалка з 20 епохами історії України та світу!

## 🚀 Швидкий старт

```bash
# Встановлення залежностей
npm install

# Запуск в режимі розробки
npm run dev

# Білд для продакшену
npm run build
```

## 📁 Структура проекту

```
project/
├── src/
│   ├── components/          # React компоненти
│   │   ├── TapArea.tsx     # Основна тапалка

```

## 🔮 Пасхалка "Sit Studio"

Збери слово **"Sit Studio"** з літер, що випадають в скринях!
- Шанс випадання: **0.1%** за скриню
- Нагорода: **x2 XP назавжди!**
- Імовірність: ~1 з 1000 активних гравців

## ⚡ Система переродження

При переродженні:
- Скидається рівень, валюта, генератори
- Зберігаються артефакти та покращення
- Відкриваються нові світові епохи

## 🛠️ Конфігурація

Створіть `.env` файл на основі `.env.example`:

```env
VITE_SUPABASE_ANON_KEY=your_key
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_TELEGRAM_BOT_USERNAME=JoltTimebot
```
